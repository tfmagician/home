cubrid_get_charset

(PECL CUBRID >= 8.3.0)

cubrid_get_charset — Returns the current CUBRID connection charset

Description

string cubrid_get_charset ( resource $conn_identifier )

This function returns This function returns the current CUBRID connection charset.

Parameters

conn_identifier

The CUBRID connection. If the connection identifier is not specified, the last link opened by cubrid_connect() is assumed.

Return Values

A string that represents the CUBRID connection charset; on success.

FALSE on failure.

Examples

Example #1 cubrid_get_charset() example

<?php
    $link = cubrid_connect("localhost", 30000, "demodb2", "dba", "");
    if (!$link)
    {
        die('Could not connect.');
    }

    printf("CUBRID current charset: %s\n", cubrid_get_charset ($link));
?>

The above example will output:

Result:
CUBRID current charset: iso8859-1
